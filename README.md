## Deploy to Vercel

```bash
git init
git add .
git commit -m "MI AI v2"
git branch -M main
git remote add origin https://github.com/YOUR/mi-ai.git
git push -u origin main
```
Then import on vercel.com → Deploy done!

## GitHub Secrets needed
- `VERCEL_TOKEN`
- `VERCEL_ORG_ID`  
- `VERCEL_PROJECT_ID`
