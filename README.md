## 🚀 Deploy to Vercel

### Step 1: Push to GitHub
```bash
git init
git add .
git commit -m "MI AI - Initial commit بِسْمِ اللَّهِ"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/mi-ai.git
git push -u origin main
```

### Step 2: Deploy on Vercel
1. Go to [vercel.com](https://vercel.com)
2. Click **"New Project"**
3. Import your GitHub repo
4. Click **Deploy** — Done! 🎉

### Step 3: GitHub Actions (24/7)
Add these secrets in GitHub → Settings → Secrets:
- `VERCEL_TOKEN` — from vercel.com/account/tokens
- `VERCEL_ORG_ID` — from `.vercel/project.json`
- `VERCEL_PROJECT_ID` — from `.vercel/project.json`

```
Project: ramadan-2385b
Auth: Email/Password + Google enabled
Database: Realtime Database
Storage: Firebase Storage
```

Enable in Firebase Console:
1. **Authentication** → Sign-in methods → Enable Email/Password + Google
2. **Realtime Database** → Create database → Start in test mode
3. **Storage** → Get started
